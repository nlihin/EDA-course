תיאור הפרויקט:¶

מערך הנתונים כולל מידע דמוגרפי של סטודנטים, הרגלי השימוש שלהם במדיה החברתית ומדדים הקשורים להשפעות השימוש על בריאות נפשית, תפקוד אקדמי ועוד. מטרת הפרויקט היא לחקור כיצד רמות השימוש במדיה החברתית משפיעות על היבטים שונים בחייהם ולזהות דפוסים וקשרים בין המשתנים. הניתוח יתבצע באמצעות טכניקות סטטיסטיות, המחשות גרפיות ובחינת קורלציות במטרה לחשוף מגמות ותובנות משמעותיות לגבי התמכרות והשפעותיה.

נוסיף עמודה חדשה של קבוצת גיל ומגדר
# יצירת קבוצות גיל-מגדר
def classify_group(row):
    if row['Age'] >= 18 and row['Age'] <= 21:
        return 'Female 18-21' if row['Gender'] == 'Female' else 'Male 18-21'
    elif row['Age'] > 21 and row['Age'] <= 25:
        return 'Female 21-25' if row['Gender'] == 'Female' else 'Male 21-25'
    else:
        return None  # מחוץ לטווח

# הוספת העמודה
df['Age_Gender_Group'] = df.apply(classify_group, axis=1)
כעת בשביל לבדוק בצורה אמינה איזה קבוצה של אנשים מוגדרת כמכורים נבדוק את אחוז המכורים מכל קבוצה ביחס לכל הקבוצה. כלומר מה אחוז האנשים בכל קבוצה שקיבלו ציון מעל 7 ברמת המתכרות
